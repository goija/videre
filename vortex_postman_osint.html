AUTHORIZATION PROTOCOLS

No Auth

Postman won't send authorization details. Select 'No Auth' if request doesn't require authorization.

API Key

Send a key-value pair in headers or query params. Store values in variables for extra security.

Bearer Token

Authenticate using an access key, such as a JWT. Cryptic string included in request header. Postman appends token to the text Bearer <Your API key>.

JWT Bearer

JSON Web Token encoded and digitally signed. Select Payload, Secret (Base64), or Private Key (RS, ES, PS algorithms in PKCS #8 format).
